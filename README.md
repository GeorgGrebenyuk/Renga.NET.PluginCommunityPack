# Renga.NET.PluginCommunityPack

Частичная копия Renga.NET.PluginUtility с фиксированными пространствами имен, включая удаленные элементы. Плюс методы расширения для работы с API

Необходимость в подобной библиотеке возникла, когда влекомые черт пойми чем, разработчики Renga Software ударились в массовые правки стандартной библиотеки Renga.NET.PluginUtility.dll из своего SDK - стали удалять "неподдерживаемые" типы, добавлять новые, менять названия пространств имен, перемещать элементы между разными классами. Эти изменения полностью разрушают принцип обратной совместимости плагинов и иначе как, вредительством, это не назвать! 

В общем, данная библиотека классов сдержит постоянную структуру перечислений, вспомогательных классов, как было с ранних версий Renga API. Библиотека не содержит символы, относящиеся к регистрации плагина, т.к. есть риск потери части содержимого при декомпиляции сборки. Напротив - перечислениями и константами можно пользоваться свободно. Если каких-либо элементов не хватает - делайте PR или просите добавить. Человеческой версионности SDK у них тоже нет, changelog не всё содержит.
